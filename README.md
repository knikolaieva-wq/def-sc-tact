# Базовый проект для TON

Этот репозиторий содержит минимальную структуру папок для старта разработки смарт‑контрактов под TON.

## Структура
- `contracts/` — исходники смарт‑контрактов (например, на Tact/FunC)
- `tests/` — тесты и вспомогательные фикстуры
- `scripts/` — скрипты сборки, деплоя, утилиты

## Дальшие шаги (опционально)
- Установить инструменты разработки:
  - Компилятор Tact: https://github.com/tact-lang/tact
  - Или toolchain для FunC/TVM, если используете FunC
- Добавить контракт в `contracts/`
- Написать скрипты сборки/деплоя в `scripts/`
- Добавить тесты в `tests/`

## Примечание
Папки содержат файл `.gitkeep`, чтобы Git отслеживал их даже когда они пустые. Удалите эти файлы, когда добавите реальные артефакты.
